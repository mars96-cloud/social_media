# AI趣创社社媒工作区

这个工作区用于管理 `AI趣创社` 的长期内容生产、增长实验、产品设计和知识沉淀。

它有两个目标：

1. 服务当前账号的长期运营。
2. 后续裁掉运营层后，作为可复制给别人的中文社媒工作区母版。

## 顶层结构

- `00_收件箱`：临时输入区
- `01_品牌系统`：定位、人群、风格、平台规则
- `02_内容系统`：内容支柱、选题规则、钩子标题、生产流程
- `03_增长系统`：流量入口、承接动作、转化路径、增长实验
- `04_产品系统`：免费资料、低客单、课程、训练营、企业服务
- `05_素材资产库`：提示词、参考案例、案例拆解、视觉素材、检查清单
- `06_运营中心`：当前月份的日历、选题池、草稿、已发布、数据记录
- `07_复盘总结`：有效模式、失效模式、用户问题、月度洞察
- `08_模板库`：内容、复盘、产品模板
- `09_归档区`：暂不活跃但需要保留的内容

## 使用原则

- 长期稳定的方法写进系统层。
- 当月执行内容放进运营层。
- 复盘出来的规律再回写到系统层。
