# 快速开始

## 每周怎么用

1. 把灵感和原始材料先放进 `00_收件箱`
2. 把值得做的内容移入当月 `选题池.md`
3. 用 `08_模板库/内容简报模板.md` 写内容简报
4. 在当月 `草稿区` 写平台草稿
5. 发布后把最终版本和结果记录到 `已发布`、`数据记录`
6. 每周、每月把有效规律回写到 `07_复盘总结` 和系统层

## 新内容的标准动作

1. 先看 `02_内容系统/选题规则.md`
2. 判断它属于哪个内容支柱
3. 明确本条内容的目标和承接动作
4. 决定做小红书版、公众号版，还是双平台联动版

## 每月要做的事

1. 完成本月 `月度复盘.md`
2. 更新 `07_复盘总结`
3. 把长期有效规则改写回 `01` 到 `04`
4. 把过期文件移入 `09_归档区`
