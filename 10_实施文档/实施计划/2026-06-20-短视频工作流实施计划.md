# 短视频工作流实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在现有社媒工作区内新增一套和图文链路彻底区分的短视频项目工作流。

**Architecture:** 以工作区模板为主，独立 skill 为辅。短视频使用单项目目录承载所有中间产物、审核状态和发布记录，不进入现有图文草稿单文件结构。

**Tech Stack:** Markdown、现有工作区目录规范、本地 Codex skill 结构

## Global Constraints

- 保持在同一 workspace 下运行，但目录、素材、状态、草稿管理必须和公众号/图文分开。
- 短视频工作流必须支持人工审核卡点，不能设计成默认全自动跑完。
- 所有新增文档和模板优先使用中文命名。
- 默认只落本地，不包含真实平台发布动作。
