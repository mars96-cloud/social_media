# 提示词模板

## 基本原则

- 每张图单独生成
- 必须通过 `$ark-image-generation` 执行
- 不要把多张图拼一张

## 单张提示词模板

```text
请生成一张 16:9 横版中文正文配图。

风格要求：
纯白背景，黑色手绘线稿为主，大量留白，少量红色、橙色、蓝色中文手写批注。整体像怪诞但清爽的产品草图，不像 PPT，不像商业插画，不像儿童插画，不要复杂背景，不要渐变，不要阴影，不要科技感 UI。

主角要求：
主角必须是“土豆崽”。土豆崽是暖土豆黄色，白色圆眼，小黑点瞳孔，厚黑眉，小黑线嘴，脸型稳定，气质是呆、轴、认真、带一点坏，不可爱，不卖萌。土豆崽必须承担画面的核心动作，不能只是装饰。

角色一致性要求：
请对齐这些本地参考锚点的角色一致性：
- assets/potato-ip/images/01_角色设定主图_定版.png
- assets/potato-ip/images/02_平静直视.png
- assets/potato-ip/images/03_微坏笑.png
- assets/potato-ip/images/04_生气皱眉.png
- assets/potato-ip/images/00_土豆崽20张总览.png
这是一只固定角色，不是每次重新设计的新土豆角色。

如果这次要做新的动作、新的表情、新的场景关系，还要参考：
- assets/potato-ip/prompts/01_角色设定主图.md
- assets/potato-ip/prompts/02_平静直视.md
- assets/potato-ip/prompts/03_微坏笑.md
- assets/potato-ip/prompts/04_生气皱眉.md
这些 prompt 是角色模板基线，不必逐字照搬，但必须用来约束角色不漂移。

主题：
{主题}

结构类型：
{结构类型}

核心意思：
{核心意思}

画面构成：
{具体画面描述}

建议元素：
{元素1} / {元素2} / {元素3} / {元素4}

建议中文标注词：
{标注1} / {标注2} / {标注3} / {标注4} / {标注5}

补充约束：
一张图只表达一个核心结构。主体控制在画面 40% 到 60% 左右。至少保留 35% 留白。不要左上角类型标题。不要做正式流程图。不要做大段解释文字。不要画成可爱海报。
```

## 改图提示词

### 去掉左上角标题

```text
请编辑这张图，只删除左上角的标题文字和下划线，其余内容保持不变。背景补成同样的干净白底，不要新增任何其他文字或图形。
```

### 增强土豆崽参与感

```text
请保持这张图的核心意思不变，但让土豆崽成为真正推动结构运转的主体。现在它太像装饰。重生成时保持白底、手绘、留白、少字、不要可爱。
```
