# 土豆崽 IP

## 角色定义

土豆崽是这套正文配图 skill 的固定主角。

它不是装饰物，不是贴纸，不是陪衬，而是要亲自承担画面里的核心动作。

## 角色气质

土豆崽的气质应当是：

- 呆
- 轴
- 稍微有点凶
- 带一点坏
- 认真做荒诞的事

不能变成：

- 软萌吉祥物
- 幼态卡通
- 潮玩
- 过度可爱角色

## 角色外观

- 土豆黄色主体
- 轻微体积感
- 白色圆眼
- 小黑点瞳孔
- 厚黑眉
- 小黑线嘴
- 脸型稳定
- 颜色稳定

## prompt 模板基线

`assets/potato-ip/prompts/` 里的 prompt 不是要求你每次原封不动照抄。

但它们是角色基线模板。

这意味着：

- 新动作可以做
- 新表情可以做
- 新场景可以做
- 新正文配图也可以做

但是：

- 角色脸型不能漂
- 角色气质不能漂
- 颜色不能漂
- 角色边界不能漂

所以只要做新的角色动作或新的场景关系，就必须先参考这些 prompt 模板。

## 土豆崽适合承担的动作

- 搬
- 拉
- 推
- 压
- 守
- 接
- 分流
- 筛选
- 承接
- 被卡住
- 从混乱里整理出路径

## 判断标准

如果去掉土豆崽，画面的意思完全不变，说明土豆崽只是装饰，必须重做。

如果连续两张图里的土豆崽看起来像两个不同角色，也算失败，必须重新校准锚点图。
