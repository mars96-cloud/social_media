# 土豆崽 IP 定版说明

## 定版目标

这一版以 `v3` 的脸型和体块作为母版，吸收：

- `v2` 的厚眉和“呆、直愣、盯着看”的眼神
- `v4` 的干净轮廓和更统一的简洁感

最终目标是做成一套可以长期复用的 `土豆崽 IP` 图库，后续无论做正文配图、封面、头像还是社媒衍生图，都以这一版为统一标准。

要避免后续再次出现这些问题：

- 主图和表情图颜色不一致
- 主图和表情图眉毛粗细不一致
- 眼睛比例漂移
- 嘴型风格乱跳
- 脸型越画越瘦、越画越圆、越画越写实
- 表面出现奇怪斑点、毛孔感、脏颗粒感

## 角色定版规则

### 1. 脸型和体块

- 整体以 `v3` 的体块为基础
- 上窄下宽，但过渡要圆钝，不要尖
- 体型偏厚实、笨、稳
- 左右边缘允许有轻微自然起伏，但不要夸张葫芦感
- 顶部不能出现尖点、芽点、触角状突起

### 2. 颜色

- 主体颜色统一为暖土黄、偏土豆肉色
- 阴影统一为稍深一点的棕黄色，不偏红、不偏灰
- 所有图保持同一色系，不允许一张偏黄、一张偏红、一张偏灰
- 表面默认干净，不做明显斑点、麻点、毛孔、写实土豆皮

说明：

- 如果后续需要增加一点“土豆特征”，只能做非常轻微、非常克制的凹坑或浅麻点
- 这些特征必须服务于识别感，不能破坏整体简洁感

### 3. 明暗和质感

- 保留轻微三维体积感
- 不做写实高光
- 不做 glossy 质感
- 用简洁明暗分块表达体积
- 整体仍然偏简约 IP，而不是写实食物

### 4. 五官

- 眼睛：两只白色圆眼，大小固定，间距稳定
- 瞳孔：小黑点，可通过位置变化表达情绪，但大小体系要稳定
- 眉毛：偏粗、黑色、干净，厚度统一，只允许角度变，不允许风格跳
- 嘴巴：统一细黑线风格，小而克制，不做夸张大嘴

### 5. 气质

核心气质不是软萌，而是：

- 呆
- 轴
- 笨
- 稍微有点凶
- 带一点坏
- 有轻微攻击感

要求：

- 不做 emoji 式夸张表情
- 不做儿童向软萌表情
- 不做写实肌肉式情绪表达

### 6. 输出约束

默认角色定版输出：

- 白底
- 单角色居中
- 无四肢
- 无衣服
- 无道具
- 无文字
- 无多余背景装饰

如果是进入正文配图或场景插图，再按任务加入动作、场景、道具。

## 用于正文配图时的角色要求

如果后续把这个 IP 用在正文配图 skill 里，角色要满足：

- 可以进入怪诞、简约、白底手绘式场景
- 可以承担“搬、拉、推、压、盯、守、卡住、筛选、承接”这类概念动作
- 允许保留轻微冷幽默感
- 不能变成过于卡通吉祥物

## 禁止项

- 不要加头顶芽点
- 不要做强写实土豆皮
- 不要做密集麻点
- 不要做幼态卖萌版
- 不要做商业潮玩质感
- 不要做复杂服装版
- 不要随意改变脸型比例
