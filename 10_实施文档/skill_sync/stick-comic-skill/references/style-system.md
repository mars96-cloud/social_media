# Stick Comic Style System

## Style Summary

This style is an original, thumbnail-first cartoon explainer system inspired by the communication pattern of popular stick-figure finance channels.

It should feel:
- blunt
- funny
- direct
- visual-first
- contrast-heavy
- easy to understand instantly

It should not feel:
- painterly
- editorial
- soft
- realistic
- decorative
- text-heavy

## Visual DNA

### 1. Character DNA
- circle head
- dot or bean eyes
- simple eyebrow shapes
- open-mouth shock, worried mouth, or smug smile
- thin stick limbs or tube limbs
- one-color shirt block
- body used as icon, not anatomy

### 2. Object DNA
Prefer oversized symbolic objects:
- cash bundles
- crown
- lock or cage
- phone screen
- stock or bond labels
- AI robot head
- path sign
- treasure chest
- hourglass or clock
- chart arrow

### 3. Page DNA
- strong empty background
- one dominant headline zone
- one central metaphor
- one emotional read
- one bottom reinforcement zone if needed

## Cover Recipes

### Recipe A: Wrong vs Right
- left side = pain, confusion, stuck state
- right side = gain, clarity, structured path
- title zone across top
- money or asset signal near right side

### Recipe B: Trap Scene
- main character stuck in cage, maze, or low-pay loop
- bait label visible
- one line hook on top

### Recipe C: Future Shock
- robot, AI, market pressure, disappearing jobs
- scared learner vs calm guide
- high urgency title

## Caption Tone

For Xiaohongshu, dialogue should be:
- short
- judgment-first
- a little sharp
- not polite filler-heavy
- not motivational soup

Preferred line length:
- 6 to 16 Chinese characters per bubble

## Safe Originality Rule

Never copy:
- exact thumbnail titles
- exact character outfit combinations if distinctive
- exact channel mascot details
- exact pose or exact scene replication

Always transform into:
- a new topic
- a new contrast
- a new wording set
- a new scene arrangement
