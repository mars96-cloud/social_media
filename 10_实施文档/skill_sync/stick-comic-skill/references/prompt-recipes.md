# Stick Comic Prompt Recipes

## Universal Base Prompt

```text
A bold original cartoon explainer illustration in a stick-figure finance-thumbnail style, white background, thick black outlines, flat bright colors, simple round-head characters, exaggerated expression, strong metaphor, clean composition, highly readable, playful but sharp, no watermark, leave space for local title overlay, do not render long Chinese text.
```

## Cover Prompt Template

```text
[base prompt]
Main topic: [topic]
Core judgment: [judgment]
Scene type: [wrong vs right / trap / future shock]
Main character emotion: [worried / shocked / smug / trapped]
Symbol objects: [cash / cage / robot / chart / crown / signs]
Composition: strong title area at top, central conflict in middle, simple bottom area, mobile-thumbnail readability.
```

## Dialogue Page Prompt Template

```text
[base prompt]
Comic page for Xiaohongshu, portrait ratio 3:4.
One simple scene only.

```
