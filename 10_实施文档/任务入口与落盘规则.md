# 任务入口与落盘规则

这份文档定义总控 skill 在工作区中的常用任务入口，以及每种任务的默认落盘位置。

## 月目录标准

统一使用：

`06_运营中心/<年份>/<YYYY-MM>/`

月目录第一层固定包含：

- `小红书/`
- `公众号/`
- `视频/`
- `月度选题/`
- `文章总览/`

其中：

- `月度选题/` 负责本月选题池、排期、系列规划
- `文章总览/` 负责母文章、母模板、统一观点稿
- 平台内容从 `文章总览/` 派生后，分别落到平台目录

## 任务入口

### 1. 写新内容

适用说法：

- 写一篇小红书
- 写一篇公众号
- 写一个视频脚本
- 围绕某个主题出一条内容

默认动作：

1. 先确认本月选题与平台目标
2. 如需跨平台复用，先产出母文章
3. 再生成平台成稿
4. 最后写入对应平台目录

默认落盘：

- 小红书：`06_运营中心/<年份>/<YYYY-MM>/小红书/`
- 公众号：`06_运营中心/<年份>/<YYYY-MM>/公众号/`
- 视频：`06_运营中心/<年份>/<YYYY-MM>/视频/`

如该主题需要作为统一源稿保留，先补一份到：

- `06_运营中心/<年份>/<YYYY-MM>/文章总览/`

### 2. 多平台改写

适用说法：

- 把这篇公众号改成小红书
- 把这条小红书改成公众号
- 把一个主题拆成多平台版本

默认动作：

1. 先抽出母稿核心观点
2. 如无母稿，先在 `文章总览/` 建立母文章
3. 再分平台改写
4. 每个平台单独保存

默认落盘：

- 母稿：`06_运营中心/<年份>/<YYYY-MM>/文章总览/`
- 公众号：`06_运营中心/<年份>/<YYYY-MM>/公众号/`
- 小红书：`06_运营中心/<年份>/<YYYY-MM>/小红书/`
- 视频：`06_运营中心/<年份>/<YYYY-MM>/视频/`

### 3. 发布前检查

适用说法：

- 帮我检查这篇能不能发
- 收一收口
- 看看标题、CTA、标签是否合理

默认动作：

1. 判断平台
2. 给出主要问题
3. 生成修订版
4. 回写原目录

默认落盘：

- 原文件所在目录
- 如无原文件，则按平台落到对应平台目录

### 4. 准备发布稿

适用说法：

- 帮我准备公众号发布稿
- 帮我整理成可发的小红书版本
- 帮我准备视频发布文案

默认动作：

1. 整理最终正文
2. 整理标题
3. 整理标签、导语、封面说明或配图说明
4. 保存到平台目录中的正式项目文件夹

默认落盘：

- `06_运营中心/<年份>/<YYYY-MM>/小红书/<内容目录>/`
- `06_运营中心/<年份>/<YYYY-MM>/公众号/<内容目录>/`
- `06_运营中心/<年份>/<YYYY-MM>/视频/<内容目录>/`

### 5. 月度选题与排期

适用说法：

- 帮我规划这个月的选题
- 给我拆 25 个选题
- 生成本月内容安排

默认动作：

1. 明确月主题
2. 输出选题池
3. 输出系列拆解和排期建议
4. 回写到本月选题目录

默认落盘：

- `06_运营中心/<年份>/<YYYY-MM>/月度选题/`

### 6. 母文章沉淀

适用说法：

- 先写母稿
- 先给我做一个统一版本
- 以后小红书和公众号都从这篇改

默认动作：

1. 明确核心观点
2. 形成统一文章结构
3. 标注可拆成哪些平台版本

默认落盘：

- `06_运营中心/<年份>/<YYYY-MM>/文章总览/`

### 7. 复盘回写

适用说法：

- 帮我复盘这条内容
- 总结这篇为什么有效
- 把经验沉淀到系统里

默认动作：

1. 提取表现结论
2. 提取结构经验
3. 提取可复用表达
4. 回写系统文档

默认落盘：

- `07_复盘总结/`
- 如涉及长期方法论，再回写 `02_内容系统/` 或 `12_平台执行框架/`
