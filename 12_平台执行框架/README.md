# 平台执行框架

这里是当前工作区的执行层规则包。

它和战略层分开：

- 战略层回答：写什么、为什么写、写给谁、承接什么
- 执行层回答：怎么写得更适合平台、怎么排版、怎么配图、怎么发前检查

## 使用方式

总控 skill 先做战略判断，再按平台读取这里的对应规则文件。

## 和工作流的关系

这一层只回答一件事：

- 内容已经确定要写了，接下来怎样写得更符合平台

它不负责决定：

- 这条内容该不该写
- 应该服务谁
- 当前阶段目标是什么
- 应该保存到哪里

这些由工作区战略层和总控 skill 中的 workflow 文档决定。
