# 公开资料与案例索引

这份索引用于后续直播、海报、资料包、私域答疑和课程延伸阅读。

## OpenAI Platform Docs - Agents
- 类别：官方学习资料
- 地址：https://platform.openai.com/docs/guides/agents
- 适合用途：理解 agent 的基本构成、工具调用与多步骤执行思路
- 课程里的讲法：适合做 agent 基础认知，不适合直接当作变现课程本体。

## IBM Think - Large language models
- 类别：官方学习资料
- 地址：https://www.ibm.com/think/topics/large-language-models
- 适合用途：解释什么是大模型、为什么能做生成与推理
- 课程里的讲法：适合做大众认知解释，语言相对友好。

## blacktwist/social-media-skills
- 类别：公开 GitHub skill
- 地址：https://github.com/blacktwist/social-media-skills
- 适合用途：快速做内容策略、文案写作、复盘分析 demo
- 课程里的讲法：优势是上手快；短板是效果依赖输入质量，不能替代内容训练。

## jihe520/social-push
- 类别：公开 GitHub workflow
- 地址：https://github.com/jihe520/social-push
- 适合用途：演示多平台内容暂存、草稿发布和社媒推送 workflow
- 课程里的讲法：适合展示流程自动化，不适合承诺最终流量结果。

## autoclaw-cc/xiaohongshu-mcp-skills
- 类别：公开 GitHub skill
- 地址：https://github.com/autoclaw-cc/xiaohongshu-mcp-skills
- 适合用途：演示小红书搜索、选题、发布、互动等动作
- 课程里的讲法：适合做平台 demo；真正账号增长仍要靠内容和反馈迭代。

## agent-browser
- 类别：公开浏览器自动化工具
- 地址：https://www.npmjs.com/package/agent-browser
- 适合用途：演示浏览器自动化、页面操作和半自动执行流程
- 课程里的讲法：适合说明 agent 可以操作网页，但不等于完整业务自动化。
