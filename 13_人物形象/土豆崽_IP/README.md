# 土豆崽 IP 资产总览

这个目录用于统一管理 `土豆崽 IP` 的角色定版、表情资产、背景资产、提示词和社媒适配图。

后续如果要基于 `$ian-xiaohei-illustrations` 一类的正文配图工作流改造成自有 IP 版本，默认优先读取这里，而不是临时重新定义角色。

## 目录结构

### 1. `images/`

角色主图、表情定版图、总览图。

建议用途：

- 正文配图的角色参考
- 角色一致性校准
- 头像、封面、社媒图的表情调用

其中建议优先使用：

- `01_角色设定主图_定版.png`
- `00_土豆崽20张总览.png`
- `social_pack/高频精选/`

### 2. `prompts/`

角色主图、表情图、背景图对应的提示词草稿与定版提示词。

建议用途：

- 后续继续扩展新表情
- 生成新背景或新姿态时保持角色一致
- 给配图 skill 提供角色描述基线

### 3. `backgrounds/`

早期或过程版背景图资产。

建议用途：

- 参考场景方向
- 参考色彩和构图思路

不建议直接默认当成最终正文配图资产。

### 4. `backgrounds_xiaohei/`

已经对齐“小黑式正文配图逻辑”的土豆崽背景实验与定稿资产。

这是后续把 `$ian-xiaohei-illustrations` 替换为土豆崽 IP 时最有参考价值的一层。

重点子目录：

- `final_clean_zh/`：较适合直接参考的中文清爽定稿
- `final_notext/`：适合后续本地排字
- `prompts/`：对应背景主题提示词

### 5. `social_pack/`

社媒使用成品与适配版，包括：

- 主页头图适配版
- 头像背景搭配预览
- 高频精选

建议用途：

- 公众号封面横版参考
- 小红书主页头图参考
- 朋友圈背景、视频号背景参考

### 6. `定版说明.md`

角色定版规则，是后续所有生图与改图的角色主规范。

## 后续使用规则

如果后面要做“正文配图 skill + 自有 IP”路线，默认按下面顺序取参考：

1. `定版说明.md`
2. `images/01_角色设定主图_定版.png`
3. `images/00_土豆崽20张总览.png`
4. `social_pack/高频精选/`
5. `backgrounds_xiaohei/final_clean_zh/`

## 角色调用原则

后续所有基于土豆崽的正文配图、封面图、信息图、插画，不要重新发明角色。

默认要求：

- 保持同一脸型与主色系
- 保持同一眼睛比例与眉毛厚度
- 保持同一“呆、轴、轻微攻击感”的气质
- 不引入写实皮肤、明显颗粒、拟人衣服或复杂配饰
