# 已发布区说明

这里存放当月已经定稿、已暂存或已发布的内容归档。

## 目录结构

每条内容单独一个目录：

- `YYYYMMDD_平台_主题/`

目录内固定包含：

- `发布稿.md`
- `配图方案.md`

后续如有需要，可追加：

- `发布链接.md`
- `数据复盘.md`

## 进入条件

一条内容进入这里，至少满足下面之一：

- 已经完成最终成稿
- 已经整理成可暂存版本
- 已经真正发布，需归档保存

## 使用原则

- 这里放的是“可交付版本”，不是随手草稿。
- 如果只是过程稿，优先放在 `草稿区`。
- 发布后的经验，再回写到 `07_复盘总结` 或系统层文档。
