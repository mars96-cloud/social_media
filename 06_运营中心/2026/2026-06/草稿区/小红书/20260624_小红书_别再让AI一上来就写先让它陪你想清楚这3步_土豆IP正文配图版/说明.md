# 说明

这是基于原稿《别再让 AI 一上来就写，先让它陪你想清楚这 3 步》改造出的土豆 IP 正文配图版。

## 本版目标

- 保留原稿核心内容
- 配图不走漫画对话页
- 改为 16:9 横版正文配图
- 风格遵循 `ian-xiaohei-illustrations` 的白底、手绘、留白、怪诞但清晰
- 视觉主角从“小黑”替换为用户自己的土豆 IP

## 土豆 IP 使用规则

- 土豆 IP 作为画面中的核心动作主体
- 不做吉祥物摆拍
- 要真的参与“思考、分流、整理、拦截、搬运、搭路”等动作
- 颜色沿用用户认可的土豆 `01` 方向

## 画面规则

- 纯白背景
- 黑色手绘线稿为主
- 少量红橙蓝中文批注
- 大面积留白
- 每张图只讲一个判断
- 不做 PPT 图
- 不做信息卡
- 不做漫画框
