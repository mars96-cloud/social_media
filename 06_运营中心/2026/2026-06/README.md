# 2026-06 运营说明

这里是 2026 年 6 月的运营执行目录。

## 目录分工

- `六月执行包.md`：本月运营重点
- `内容日历.md`：本月发布时间安排
- `选题池.md`：待写选题与备选主题
- `草稿区/`：正在写和正在改的内容
- `已发布/`：已定稿、已暂存、已发布内容
- `数据记录/`：发布后的数据记录
- `月度复盘.md`：本月阶段总结

## 推荐流转

1. 先从 `选题池.md` 或 `00_收件箱` 进入
2. 成稿前放入 `草稿区`
3. 定稿后转入 `已发布`
4. 发布后把结果记到 `数据记录`
5. 月底回收进 `月度复盘.md`
