# 运营中心目录规则

`06_运营中心` 按 `年 / 月` 管理当期内容生产与发布准备。

标准月目录结构如下：

```text
06_运营中心/<年份>/<YYYY-MM>/
├─ 小红书/
├─ 公众号/
├─ 视频/
├─ 月度选题/
└─ 文章总览/
```

## 月目录说明

- `小红书/`
  - 存放当月小红书相关的草稿、图文包、漫画版、配图、发布稿等。
- `公众号/`
  - 存放当月公众号文章、配图方案、封面、发布稿等。
- `视频/`
  - 存放当月视频项目、脚本、分镜、封面、发布准备等。
- `月度选题/`
  - 存放本月选题池、排期、选题拆解、系列规划。
- `文章总览/`
  - 存放本月的母文章、母模板、统一观点稿。
  - 后续小红书、公众号、视频等内容，优先从这里衍生。

## 执行规则

1. 先在 `月度选题/` 确定本月选题与排期。
2. 再在 `文章总览/` 形成母文章或母模板。
3. 然后按平台拆分到 `小红书/`、`公众号/`、`视频/`。
4. 平台目录下允许再按需要细分 `草稿`、`已发布`、`素材` 等子目录，但月目录第一层固定保持上面的 5 个入口。

## 兼容说明

- 历史月份里仍然可能保留旧目录，如 `草稿区/`、`短视频项目/`、`已发布/`。
- 旧目录先作为兼容层保留，后续新增内容一律优先进入新结构。
