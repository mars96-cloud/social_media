# 抖音脚本规则

## 当前定位

抖音还属于后续扩展平台，优先做的是脚本，不是直接执行发布。

## 目标

把已经验证过的图文主题改成更适合口播和短视频的表达。

## 结构

输出必须包含：

1. 主题 / 标题方向
2. 开场 3 秒钩子
3. 口播正文
4. 镜头或画面提示
5. 结尾 CTA

## 风格要求

- 更口语
- 更短句
- 更像在说，不像在写
- 开头 3 秒必须先抓人

## 选题来源

优先从这些地方改写：

- 已经验证过的小红书高反馈主题
- 已经写好的公众号方法论文章
- 当前月份执行包中的重点主题

## 当前限制

这个技能当前只负责脚本和结构，不负责实际视频拍摄与发布。
