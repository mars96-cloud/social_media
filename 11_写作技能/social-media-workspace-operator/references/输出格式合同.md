# 输出格式合同

## 默认输出顺序

除非用户明确只要最终稿，否则先输出：

1. 内容决策卡
2. 平台成稿

## 内容决策卡格式

按下面顺序输出：

1. 目标人群
2. 内容支柱
3. 当前目标
4. 选题判断
5. 主 CTA

内容决策卡写完后，才能进入平台稿。

## 小红书输出格式

按下面顺序输出：

1. 内容决策卡
2. 标题候选
3. 开头钩子
4. 正文
5. CTA
6. 标签

## 公众号输出格式

按下面顺序输出：

1. 内容决策卡
2. 标题
3. 导语
4. 正文分节
5. 结尾
6. CTA

## 抖音输出格式

按下面顺序输出：

1. 内容决策卡
2. 主题 / 标题方向
3. 开场 3 秒钩子
4. 口播脚本
5. 镜头提示
6. CTA

## 多平台改写输出格式

如果是多平台改写，先输出：

1. 内容决策卡
2. 核心主题
3. 核心结论
4. 小红书版
5. 公众号版
6. 抖音版（如果用户要求）

## 发前检查输出格式

如果是发前检查，按下面顺序输出：

1. 平台判断
2. 主要问题
3. 修改建议
4. 收口后的最终版
